# KNES 381 Final Project- Data Analysis Portfolio
## Overview
This project is a GitHub Pages Portfolio developed for KNES 381. It shows the application of computational tools and works as my own living resume. 

The website presents a combination of python-based analysis, excel calculations, and interactive scratch programs that showcase both technicala and logical problem-solving skills.

### Project Objectives
- Analyze VO2 and gas exchange data using Python
- Creates dynamic, programmatic data visualizations
- Demonstrate reproducible analysis through coding
- Apply Excel for statistical calculations and data organization
- Develop a logic-based interactive program
- Present all components in a clear, structured web portfolio
