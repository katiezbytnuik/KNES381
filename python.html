Methods

This analysis was performed on Kaggle Notebook Editor using Python. The dataset was imported and cleaned by renaming variables such as VO2, ventilation (VE), and time. The program then processes these variables and genrates a graph representing the physiological data I have inputted. This code is reproducible and followed Dr. John Holash's Demo-2 on Kaggle Notebook. By changing the dataset all graphs will be automatically updated.

Interpretation

The graph shown in Figure 1. shows that VO2 increases steadily as exercise continues, indicating a greater oxygen demand as workload increases. Ventilation also increases, with a steeper rise during higehr intensities, this shows how the respiratory system responds to increased metabolic demand.

the graph in Figure 2. shows that VCO2 rises with VO2, while FECO2 increases and then begins to change during higher intensities. This shows the changes in gas exchange during exercise and helps to demonstrate how physiological responses shift as the intensity increases.
